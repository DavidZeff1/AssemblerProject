unParsedProg holds raw data entered by user before being parsed for macros empty lines and comments,  macroparser.c will parse the unparsedProg for comments newlines and macros and will create a new file 'prog.as', firsRun.c will convert the data from prog.as to base64 machine code and if no errors occur it will output 3 files:  prog.ob prog.ent prog.ext, if there are errors it will ouput error textfile.txt
